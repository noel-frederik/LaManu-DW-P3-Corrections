Consignes du TP1 : Le juste prix

Construisez une page html avec un bouton et un champ number. Le but est de trouver un nombre entre 0 et 100.
A chaque réponse la page répond :

  • Plus
  • Moins
  • Correcte

Quand la réponse est trouvée, on obtient le nombre d'essais que l'on a fait.
